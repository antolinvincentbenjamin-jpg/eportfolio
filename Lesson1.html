Lesson 1 — Introduction to CSS

“Color is a language of the soul.” – Vincent Van Gogh

What is CSS?

CSS (Cascading Style Sheets) brings life to HTML pages by painting them with color, shape, and motion. It defines how elements appear — their layout, fonts, and tones — separating beauty from structure. Like Van Gogh’s brush, CSS turns plain markup into art.

Purpose of CSS

  • Adds visual appeal to plain HTML
  • Separates content and design
  • Ensures consistent styling across pages
  • Improves user experience and accessibility

Example

Below is a simple demo showing how CSS colors a page like paint on canvas:

<!DOCTYPE html>
<html>
<head>
<style>
body {
  background-color: midnightblue;
  color: gold;
  font-family: 'Georgia';
}
h1 {
  text-align: center;
  color: lightyellow;
}
</style>
</head>
<body>
  <h1>Hello, Starry World!</h1>
  <p>CSS makes the web shine like stars.</p>
</body>
</html>
