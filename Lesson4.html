Understanding CSS Rules

A CSS Rule consists of two parts: the selector (which element it applies to) and the declaration block (what style properties are applied). Within the declaration block, there can be one or more style properties ‚Äî each changing a visual aspect of the element. 

selector {
  property: value;
  property: value;
}

Using Class Selectors

Class selectors are used to apply styles to specific elements with a class attribute. They begin with a period (.) and can be reused multiple times.

.red {
  background-color: red;
}
</style>
