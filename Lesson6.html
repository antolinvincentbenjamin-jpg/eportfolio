Property for Dimensions

CSS allows us to manipulate the size, visibility, and proportion of elements on a webpage. Below is a summary of common dimension properties and their purposes:

Property Definition Example
visibilityControls whether an element is visible or hidden.visibility: hidden;
widthDefines the width of an element.width: 200px;
heightDefines the height of an element.height: 150px;
line-heightSets the amount of space between lines of text.line-height: 20px;
max-heightSets the maximum height allowed for an element.max-height: 120px;
min-widthSets the minimum width allowed for an element.min-width: 200px;
